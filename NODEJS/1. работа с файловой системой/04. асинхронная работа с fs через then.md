# асинхронная работа с fs через then

- с методами модуля fs асинхронно можно работать не только через коллбэки, но и через промисы
- для этого есть специальное свойство `promises`, содержащее в себе промисные аналоги методов для работы с файловой системой
- к примеру, для метода `fs.readFile` его промисный аналог будет `fs.promises.readFile`

## чтение файлов

- выведем в консоль содержимое какого-нибудь файла:

```js
fs.promises.readFile('readme.txt', 'utf8').then((data) => {
  console.log(data);
});
```

## обработка исключений

- добавим теперь обработку исключительный ситуаций

```js
fs.promises
  .readFile('readme1.txt', 'utf8')
  .then((data) => {
    console.log(data);
  })
  .catch((error) => {
    console.log('ошибка');
  });
```

## чтение и запись

- можно прочитать файл, что-то сделать с его текстом, а потом записать обратно:

```js
fs.promises
  .readFile('readme.txt', 'utf8')
  .then((data) => {
    return fs.promises.writeFile('readme.txt', data + '!');
  })
  .catch((error) => {
    console.log('ошибка');
  });
```

✅ `then` должен _возвращать_ новый промис, чтобы дальше передавать по цепочке. без `return` `then` вернёт `undefined`

### задача

- Пусть в файле через запятую записаны числа. Сделайте скрипт, который запишет каждое из этих чисел в отдельный файл.

- решение

```js
fs.promises
  .readFile('readme.txt', 'utf8')
  .then((data) => {
    const numbers = data.split(', ').map((digit) => Number(digit));
    return numbers.forEach((number) => {
      fs.promises.writeFile(`${number}.txt`, String(number));
    });
  })
  .catch((error) => {
    console.log('ошибка');
  });
```

## массовая работа

- пусть есть несколько файлов
- прочитаем эти файлы, соединим их тексты в одну строку и запишем её в новый файл

- в отличие от коллбэков, в данном случае нет нужды выполнять чтение файлов по очереди
- при работе с промисами можно записать все промисы для чтения файлов в массив, а потом воспользоваться `Promises.all`, чтобы осуществить запись в файл только тогда, когда все эти файлы будут прочитаны

- пусть имена файлов есть в виде массива:

```js
const fileNames = ['text`.txt', 'text2.txt', 'text3.txt'];
```

- запустим цикл, в котором будем читать эти файлы, записывая промисы с результатами в массив:

```js
const filePromises = [];

for (let fileName of fileNames) {
  filePromises.push(fs.promises.readFile(fileName, 'utf8'));
}
```

- имея такой массив, можно вызвать `then` только один раз, когда все промисы выполнятся:

```js
Promise.all(filePromises)
  .then((data) => {
    return fs.promises.writeFile('res.txt', data.join(''));
  })
  .catch((error) => {
    console.log('ошибка');
  });
```

### задача

- Пусть у вас есть 5 файлов с числами. Найдите сумму этих чисел и запишите в новый файл.
- решение:

```js
const fileNames = ['number1.txt', 'number2.txt', 'number3.txt', 'number4.txt', 'number5.txt'];

const filePromises = [];

fileNames.forEach((fileName) => {
  filePromises.push(fs.promises.readFile(fileName, 'utf8'));
});

Promise.all(filePromises)
  .then((fileContents) => {
    let sum = 0;
    fileContents.forEach((fileContent) => {
      sum += Number(fileContent);
    });
    return fs.promises.writeFile('result.txt', String(sum));
  })
  .catch((error) => {
    console.log('ошибка');
  });
```
