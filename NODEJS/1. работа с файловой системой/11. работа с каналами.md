# работа с каналами

- ситуация, в которой нужно прочитать данные из одного потока и записать их в другой, является достаточно распространённой
- для упрощения такой операции придумали `каналы` (pipes)

- пусть есть поток чтения и поток записи

```js
const readableStream = fs.createReadStream('readme.txt', 'utf8');
const writeableStream = fs.createWriteStream(`writeme.txt`);
```

- сделаем так, чтобы поток чтения сразу перенаправлялся в поток записи с помощью метода `pipe`:

```js
readableStream.pipe(writeableStream);
```

## цепочки

- метод `pipe` можно вызывать несколько раз цепочкой друг за другом
- при этом каждый вызов метода позволяет выполнять над данными некоторые операции:

```js
readableStream.pipe(операция).pipe(операция).pipe(операция);
```

- например выполним архивацию файла
- для этого надо сначала считать файл, затем сжать данные и потом записать сжатые данные в файл-архив
- используем встроенную в nodeJS библиотеку `zlib`

```js
import fs from 'fs';
import { createGzip } from 'zlib';

const readableStream = fs.createReadStream('readme.txt', 'utf8');
const writeableStream = fs.createWriteStream(`writeme.txt`);

readableStream.pipe(createGzip()).pipe(writeableStream);
```
