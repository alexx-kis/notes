# создание сервера на express

- существует 4 вида программ на nodeJS

1. веб-приложения (простые одностраничные приложения, микрослужбы rest и web-приложения полного стека)
2. средства командной строки (npm, gulp, webpack)
3. демоны (для запуска приложений, типа pm2)
4. десктопные приложения (строятся с применением фреймворка electron)

- для того, чтобы создавать сервера, обычно используют фреймворк **express**

```js
const express = require('express'); // подключение модуля
const app = express(); //вызов модуля express

app.get('/', (req, res) => {
  // "/" - путь и функция с запросом и ответом
  res.send('hello world!'); // send - метод отправки
});

app.listen(3000, () => {
  // прослушка порта, 3000 - порт, можно передать константу
  console.log(`express web app on: http://localhost:3000`);
});
```

## использование npm-скриптов

- в соответствующем файле package.json есть скрипты, можно через запятую добавлять любое количество скриптов

```json
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1",
    "start": "node index" // добавим скрипт для старта приложения
  },
```

- теперь можно запустить приложение командой `npm start`
