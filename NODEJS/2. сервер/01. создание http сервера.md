# создание http сервера

- развернём HTTP сервер на nodeJS, он будет принимать запросы от браузера и отдавать в ответ HTML код страниц

- импортируем модуль http:

```js
import http from 'http';
```

- после этого можно стартовать сервер с помощью метода `createServer`:

```js
http.createServer();
```

- этот метод параметром принимает коллбэк, который будет выполняться каждый раз, когда кто-то через браузер обращается к сайту:

```js
http.createServer(() => {
  //...
});
```

- в коллбэке нужно определить 2 параметра:
  1. в первый параметр попадёт объект с данными запроса пользователя
  2. во второй параметр попадёт объект, с помощью которого мы сформируем ответ, отправляемый в браузер

```js
http.createServer((request, response) => {
  //...
});
```

- с помощью метода `write` мы можем говорить, какой текст отправить в браузер:

```js
http.createServer((request, response) => {
  response.write('text1');
  response.end();
});
```
- с помощью метода `end()` мы командуем завершить ответ и отправить его в браузер

- после запуска мы должны определить, на каком порту сервер будет ожидать запросы от браузера с помощью метода `listen`, в который нужно передать порт (например, 3000):

```js
http.createServer((request, response) => {
  response.write('text1');
  response.end();
}).listen(3000);
```

- запустим скрипт
- теперь можно обратиться к серверу через браузер: http://localhost:3000

- чтобы остановить сервер, нужно использовать сочетание клавиш `ctrl/cmd+C`
