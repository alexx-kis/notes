# потоки чтения и записи

- если есть достаточно большой файл (100 МБ), и с его данными нужно что-то сделать, сначала нужно его прочитать
- в таком случае все 100 МБ файла попадут в оперативную память сервера
- если таких файлов несколько, то память сервера закончится и он упадёт

- для решения этой проблемы есть `потоки`, они позволяют прочитывать файл по частям
- для создания потока используется метод `createReadStream` (из общей части модуля `fs`):

```js
const readableStream = fs.createReadStream('readme.txt', 'utf8');
```

- теперь прочитаем файл по частям:

```js
readableStream.on('data', (chunk) => {
  console.log(chunk);
});
```

## потоки записи

- можно так же создавать потоки записи с помощью метода `createWriteStream`

```js
const writableStream = fs.createWriteStream('writeme.txt');
```

- запишем что-нибудь в потом с помощью метода `write`:

```js
writeableStream.write('text1\n');
writeableStream.write('text2\n');
writeableStream.write('text3\n');

writeableStream.end();
```

## чтение и запись

- можно прочитывать один файл, записывая по частям его данные в другой файл

```js
const readableStream = fs.createReadStream('readme.txt', 'utf8');
const writeableStream = fs.createWriteStream('writeme.txt');

readableStream.on('data', (chunk) => {
  writeableStream.write(chunk);
});
```
