# npm

- вместе с nodeJS поставляется npm (node package manager), который позволяет скачивать различные модули и плагины и подключать в проект

- чтобы проверить, что npm установлен, используется команда `npm -v`

- пакеты устанавливаются с помощью команды `npm install` или `npm i`

- чтобы начать использовать npm, нужно создать файл package.json в том каталоге, в котором находится проект

- проще всего создать этот файл через npm, а не вручную, в нём будут храниться все конфигурации:

```powershell
npm init -y
```

(без "-y" нужно будет заполнять все параметры)

- создастся файл с содержимым:

```powershell
{
  "name": "exx",
  "version": "1.0.0",
  "description": "",
  "main": "index.js",
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "keywords": [],
  "author": "",
  "license": "ISC",
  "type": "commonjs"
}
```

- установим модуль express:

```bash
npm i --save express
```

- в файле package.json появится раздел dependencies:

```json
"dependencies": {
    "express": "^5.2.1"
  }
```
- так же появится папка node_modules, в которой будут находиться все модули, необходимые для express, и файл `package-lock.json`
