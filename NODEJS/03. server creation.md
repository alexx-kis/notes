# как создать сервер на NodeJS

## модуль http

- создадим константу `http` в которую подключим модуль http:

```js
const http = require('http');
```

- таким образом в данной константе хранится весь этот модуль, и можно пользоваться его возможностями

- создадим константу для порта, порт всегда нужен для работы с сетями:

```js
const port = 8080;
```

- создаём сервер:

```js
const server = http.createServer((req, res) => {
  res.end('hello world!');
});
```

- у модуля http вызываем метод `createServer`
- `createServer` принимает параметром функцию с двумя параметрами `req` (запрос) и `res` (ответ)
- в теле функции у ответа от сервера `res` вызовем метод `end` и передадим ему строку 'hello world!'

- пока сервер работать не будет, у сервера необходимо вызвать метод `listen` и передать ему порт, на котором находится сервер:

```js
server.listen(port, () => {
  console.log(`Server listening on: http://localhost:${port}`)
})
```

- теперь можно запустить файл index.js в терминале командой `node index.js`

- при переходе по ссылке `http://localhost:8080` браузер откроет страницу, на которой будет "hello world!"

- чтобы остановить сервер, нужно в терминале нажать `cmd/ctrl+c`
