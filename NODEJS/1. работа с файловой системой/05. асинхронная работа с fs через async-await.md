# асинхронная работа с fs через async-await

- теперь вместо `then` будем использовать альтернативный синтаксис `async-await`
- в этом случае мы получим возможность обращаться с асинхронным кодом так же просто и удобно, как и с синхронным

- прочитаем текст файла и выведем его в консоль:

```js
async function func() {
  const data = await fs.promises.readFile('readme.txt', 'utf8');
  console.log(data);
}

func();

// или стрелочная функция

const func = async () => {
  const data = await fs.promises.readFile('readme.txt', 'utf8');
  console.log(data);
};

func();
```

- добавим обработку ошибок:

```js
const func = async () => {
  try {
    const data = await fs.promises.readFile('readme.txt', 'utf8');
    console.log(data);
  } catch (error) {
    console.log('ошибка');
  }
};

func();
```

- прочитаем 3 файла, соединим их текст и выведем в консоль:

```js
const func = async () => {
  try {
    const data1 = await fs.promises.readFile('number1.txt', 'utf8');
    const data2 = await fs.promises.readFile('number2.txt', 'utf8');
    const data3 = await fs.promises.readFile('number3.txt', 'utf8');

    console.log(data1 + data2 + data3);
  } catch (error) {
    console.log('ошибка');
  }
};

func();
```

- запишем текст 3 файлов в новый файл:

```js
const func = async () => {
  try {
    const data1 = await fs.promises.readFile('number1.txt', 'utf8');
    const data2 = await fs.promises.readFile('number2.txt', 'utf8');
    const data3 = await fs.promises.readFile('number3.txt', 'utf8');

    await fs.promises.writeFile('res.txt', data1 + data2 + data3);
  } catch (error) {
    console.log('ошибка');
  }
};

func();
```

- пусть имена файлов записаны в массиве, прочитаем данные файлов в цикле, а затем запишем их в новый файл:

```js
const func = async () => {
  try {
    const fileNames = ['number1.txt', 'number2.txt', 'number3.txt'];
    const data = [];

    for (let name of fileNames) {
      data.push(await fs.promises.readFile(name, 'utf8'));
    }

    await fs.promises.writeFile('res.txt', data.join(''));
  } catch (error) {
    console.log('ошибка');
  }
};

func();
```
