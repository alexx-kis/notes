# настройка ответа от сервера

- с помощью свойства `statusCode` указывается код HTTP-ответа, как правило, это код 200, сообщающий браузеру о том, что всё ок и страница найдена:

```js
http.createServer((request, response) => {

  response.statusCode = 200; // код статуса

  response.write('hello world1');
  response.end();
}).listen(3000);
```

## HTTP-заголовки

- с помощью метода `setHeader` можно отправлять HTTP-заголовки:

```js
http.createServer((request, response) => {

  response.setHeader('Content-Language', 'ru') // http-заголовок

  response.statusCode = 200;
  response.write('hello world1');
  response.end();
}).listen(3000);
```

## отправка HTML

- пока браузер трактует ответ, как обычный (plain) текст
- чтобы браузер показывал html, нужно задать соответствующий http заголовок:

```js
http.createServer((request, response) => {
  response.setHeader('Content-Language', 'ru');

  response.setHeader('Content-Type', 'text/html'); // добавим заголовок

  response.statusCode = 200;

  response.write('<b>hello world</b>'); // поменяем текст на html-тэг

  response.end();
}).listen(3000);
```

## код ответа + заголовки

- код ответа и заголовки можно объединить в одном методе `writeHead`
- первым параметром он принимает код ответа, вторым - объект с заголовками:

```js
http.createServer((request, response) => {

  response.writeHead(200, { // объединим в один метод
    'Content-Language': 'ru',
    'Content-Type': 'text/html',
  });

  response.write('<b>hello world</b>');
  response.end();
}).listen(3000);
```
