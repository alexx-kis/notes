# сокращение fs.promises

- после подключения `import fs from 'fs'` доступны промисные варианты методов через `fs.promises`

```js
fs.promises.readFile('readme.txt', 'utf8').then((data) => {
  console.log(data);
});
```
- однако, писать слово `promises` перед каждым нужным методом не удобно => запишем в переменную `fs` не сам модуль `fs`, а его часть с промисами:

`import fs from 'fs/promises`

```js
fs.readFile('readme.txt', 'utf8').then((data) => {
  console.log(data);
});
```
