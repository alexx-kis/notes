# относительные пути

- в параметр методов чтения и записи следует писать имя файла
- однако, это работает только в том случае, если читаемый файл лежит в той же папке, в которой запускается скрипт

- если файл лежит в другом месте, то в параметр метода нужно писать путь к этому файлу

### пример

    doc/
    ├── index.js
    └── directory/
        └── readme.txt

- прочитаем содержимое текстового файла
- для этого кроме имени файла нужно указать ещё и папку, в которой он лежит

```js
import fs from 'fs/promises';

const path = 'directory/readme.txt';
const data = await fs.readFile(path, 'utf8');
```

- можно использовать и абсолютный путь, если файл вообще где-то в другом месте

#### win:

`C:\Users\Alex\project\readme.txt`

или

`C:/Users/Alex/project/readme.txt`

#### mac/linux:

`/Users/alex/project/readme.txt`
