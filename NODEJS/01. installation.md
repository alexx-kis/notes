# установка и начало работы

- NodeJS - это асинхронная управляемая событиями исполнительная платформа JavaScript с мощной, но компактной стандартной библиотекой

существует 2 активно поддерживаемые версии:
1) Текущая (Current) - самая последняя версия
2) Версия с долгосрочной поддержкой (LTS - Long Term Support)

nodejs.org

- установка происходит путём скачивания файла с официального сайта

- для того, чтобы проверить, какая версия nodeJS установлена, нужно открыть терминал и выполнить команду

```bash
node -v
```

- создадим файл index.js и напишем команду 

```js
console.log('hello world');
```

- откроем терминал и выполним команду

```bash
node index.js
```

- код из файла index.js отобразится в терминале:

```bash
hello world
```
