# отдача HTML страниц из файлов

- теперь по запросу будем отдавать не какой-то текст, а корректно оформленную HTML страницу, пусть она хранится в файле page.html

```html
<!DOCTYPE html>
<html>
	<head>
		<meta charset="utf-8">
		<title>my page</title>
	</head>
	<body>
		<p>
			my first page on NodeJS
		</b>
	</body>
</html>
```

- по запросу прочитаем эту страницу из файла и отправим её в браузер:

```js
http
  .createServer(async (request, response) => {
    if (request.url !== '/favicon.ico') {
      const text = await fs.readFile('page.html', 'utf8');

      response.writeHead(200, { 'content-type': 'text/html' });
      response.write(text);
      response.end();
    }
  })
  .listen(3000);
```

- чтение файла осуществляется асинхронным образом
- в этом случае в ожидании считывания файла с диска сервер сможет обрабатывать запросы других пользователей к серверу
