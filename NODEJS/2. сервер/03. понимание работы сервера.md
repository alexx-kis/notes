# понимание работы сервера

- сервер, будучи один раз запущенным, обрабатывает запросы всех пользователей сайта
- этим подход nodeJS отличается от подхода PHP, в котором скрипт запускается, отрабатывает и останавливается

- в случае nodeJS получается, что функция-коллбэк сервера вызывается на каждый запрос
- это значит, что внешние переменные этой функции будут общими для всех запросов

- таким образом можно сделать, например, счётчик запросов к серверу и отдавать его значение каждому запросу:

```js
let i = 0;

http
  .createServer((request, response) => {
    response.writeHead(200, {
      'Content-Language': 'ru',
      'Content-Type': 'text/html',
    });
    response.write(String(++i));
    response.end();
  })
  .listen(3000);
```
