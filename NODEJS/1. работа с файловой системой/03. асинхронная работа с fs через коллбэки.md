# асинхронная работа с fs через коллбэки

- как правило, все методы модуля fs существуют в 2 вариантах - синхронном и асинхронном
- например, для синхронного чтения файла используется метод `readFileSync`, а для асинхронного - `readFile`
- аналогично для записи файла - `writeFileSync` и `writeFile`

## асинхронное чтение файла

- метод readFile принимает 3 параметра:
  1. имя или путь к файлу
  2. кодировку
  3. коллбэк, который выполнится после чтения файла

- в коллбэк следует передавать 2 параметра:
  1. объект с ошибкой, если она произойдёт
  2. текст прочитанного файла

```js
fs.readFile('readme.txt', 'utf8', (error, data) => {
  console.log(data);
});
```

## проверка асинхронности

- можно убедиться в том, что чтение файла происходит асинхронно, для этого выведем что-нибудь в консоль после работы с методом readFile:

```js
fs.readFile('readme.txt', 'utf8', (error, data) => {
  console.log(data);
});

console.log('!!!');
```

- коллбэк будет выполнен после того, как файл будет прочитан
- пока файл читается, код скрипта будет выполняться дальше
- это значит, что в консоли сначала появится результат второго `console.log`, а потом первого

## обработка исключительных ситуаций

- так как код асинхронный, то исключительные ситуации нельзя поймать через try-catch
- для обработки исключений в коллбэке предназначен первый параметр
- этот параметр будет содержать `null`, если исключения не случилось, или объект с ошибкой, если исключение произошло

- допишем код коллбэка так, чтобы он обрабатывал исключения:

```js
fs.readFile('readme.txt', 'utf8', (error, data) => {
  if (!error) {
    console.log(data);
  } else {
    console.log('ошибка', error);
  }
});
```

## асинхронная запись файла

- выполняется аналогично:

```js
fs.writeFile('readme.txt', 'какой-то текст', (error) => {
  if (error) {
    console.log('ошибка', error);
  }
});
```

## асинхронное чтение нескольких файлов

- пусть есть 2 файла с числами, найдём произведение этих чисел
- для этого нужно прочитать оба этих файла, но, так как код асинхронный, нужно читать второй файл в коллбэке первого:

```js
fs.readFile('readme1.txt', 'utf8', (error, data1) => {
  if (!error) {
    fs.readFile('readme2.txt', 'utf8', (error, data2) => {
      if (!error) {
        console.log(data1 * data2);
      } else {
        console.log('ошибка чтения файла readme2');
      }
    });
  } else {
    console.log('ошибка чтения файла readme1');
  }
});
```

## асинхронное чтение и запись файла

- предположим нужно прочитать файл, сделать с ним текстовую операцию и записать обратно в этот или другой файл
- в этом случае запись в файл нужно будет делать в коллбэке чтения

```js
fs.readFile('readme.txt', 'utf8', (error, data) => {
  if (!error) {
    fs.writeFile('readme.txt', data + '!', (error) => {
      if (error) {
        console.log('ошибка записи файла');
      }
    });
  } else {
    console.log('ошибка чтения файла');
  }
});
```
