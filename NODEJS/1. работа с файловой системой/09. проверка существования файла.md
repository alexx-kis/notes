# проверка существования файла

- в современном nodeJS проверка существования файла делается особым образом через метод `fs.access`, проверяющий возможность доступа к файлу
- импортируем асинхронную часть модуля `fs` и специальные константы из общей части модуля `fs`:

```js
import fs from 'fs/promises';
import { constants } from 'fs';
```

- после этого в переменной `constants` будет специальный объект с константами, содержащими специальные флаги, которые мы будем передавать параметром в метод `access`, изменяя его поведение

- флаг для проверки существования файла:

```js
constants.F_OK;
```

- проверим с помощью этого флага существование файла test.txt:

```js
fs.access('test.txt', constants.F_OK);
```

- метод `access` своим результатом возвращает промис
- если файл существует, то промис зарезолвится, а если нет - зареджектится:

```js
fs.access('test.txt', constants.F_OK)
  .then(() => {
    console.log('file exists');
  })
  .catch(() => {
    console.error('file does not exist');
  });
```

- перепишем код через `await`, для этого обернём вызов `access` в `try-catch`

```js
try {
  await fs.access('test.txt', constants.F_OK);
} catch (error) {
  console.log('file does not exist');
}
```

## другие флаги

- флаг для проверки разрешения на чтение файла: `constants.R_OK`

- флаг для проверки разрешения на запись файла: `constants.W_OK`
