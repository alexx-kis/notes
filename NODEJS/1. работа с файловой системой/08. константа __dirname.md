# константа \_\_dirname. имя папки со скриптом

- если nodeJS работает в стиле CommonJS, то в файлах со скриптами будет доступна константа `__dirname`

- в ES модулях эта константа была убрана, но её можно получить самому
- создадим файл `__dirname.js`, экспортирующий нужный путь к папке со скриптом:

```js
import { dirname } from 'path';
import { fileURLToPath } from 'url';

const __dirname = dirname(fileURLToPath(import.meta.url));

export default __dirname;
```

- теперь в нужно исполняемом файле можно импортировать созданный нами модуль и получить нужную константу:

```js
import __dirname from './__dirname.js';
console.log(__dirname);
```
